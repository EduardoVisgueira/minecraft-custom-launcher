COMO USAR O CONFIG PACK DO FANCYMENU
=====================================

1. Certifique-se que o mod FancyMenu está no modpack (versão 1.20.1 compatível).
   Download: https://www.curseforge.com/minecraft/mc-mods/fancymenu

2. Copie a pasta "fancymenu/" (que está dentro deste diretório) para dentro da
   pasta ".minecraft/" do servidor/modpack.

3. Coloque suas imagens de slideshow dentro de:
   fancymenu/assets/slideshow/
   Nomeie como: bg1.png, bg2.png, bg3.png (adicione mais se quiser e edite o JSON)

4. Para editar o layout do menu, edite o arquivo:
   fancymenu/customizations/main_menu.json

5. O botão de Singleplayer já está removido por padrão.
   Para reativá-lo, remova "singleplayer" da lista "hide_vanilla_buttons" no JSON.

6. Este config pack é distribuído automaticamente pelo auto-updater do launcher
   quando você incluir os arquivos no manifest.json do modpack.
